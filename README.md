# GLOWL

Cinematic production site for brands, games and entertainment.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Deploy (GitHub Pages)

Pushes to `main` deploy automatically via GitHub Actions.

Live URL: `https://lisenbart.github.io/glowl/`

## Production build

```bash
npm run build
npm run preview
```
